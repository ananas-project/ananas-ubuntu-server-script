/**********************************
 * how to use this script-tool-kit
 **********************************/

1. install git on the server with follow command:

	$ sudo apt-get install git

	or

	$ apt-get install git

2. clone this respository from githut.com:

	$ git clone https://github.com/ananas-project/ananas-ubuntu-server-script.git auss

3. use the scripts in directory:
	
	 ./auss/aus-script/
	 
	 to admin the server.
	 
4. to update this repository with script

	./auss/aus-script/pull-self.sh
